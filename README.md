# 📚 Al-Bawabah Website

**Al-Bawabah** is an educational platform designed to help Moroccan students prepare for exams and improve their study experience.  
It provides organized resources, revision materials, and tools to support learners across different levels.

---

## 🎯 Purpose

The goal of this project is to:
- Simplify exam preparation for Moroccan students
- Centralize study resources in one place
- Help students revise efficiently and consistently
- Support subjects taught in Moroccan schools

---

## ✨ Features

- 📖 Organized study materials by subject and level  
- 📝 Exam preparation resources (practice & revision content)  
- 🧠 Easy navigation for quick access to topics  
- 📱 Responsive design (works on mobile and desktop)  
- 🔎 Simple structure for fast learning
